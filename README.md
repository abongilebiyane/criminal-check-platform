# Criminal Check Platform

A web-based platform for criminal record verification.  
Built with **Node.js**, **Express**, **PostgreSQL**, and **React**.

---

## 🚀 Features
- Secure backend with Node.js and Express
- PostgreSQL database integration
- REST API endpoints for record verification
- React frontend (coming soon)
